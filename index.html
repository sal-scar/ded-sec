Requirements

  • Device: Android with Termux installed.
  • Storage: Min 3GB free. (Radio takes more storage; images/recordings also consume space.)
  • RAM: Min 2GB .

Step-by-Step Installation

  1. Install Termux & Add-Ons
    Note: To install APKs (e.g., F-Droid), ensure:
    - Enabled unknown sources (Settings > Security > Install Unknown Apps ).
    - Download F-Droid, get Termux from F-Droid .
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow fdeokd process when prompted.
  2. Run Setup Commands
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. Clone Repository
    git clone https://github.com/dedsec1121fk/DedSec
    Tip: If you get his error "Traceback (most recent call last): File "/data/data/com.termux/files/home/DedSec/Scripts/Settings.py", line 7, in import requests ModuleNotFoundError: No module named 'requests' " use this command and try again:
    pip install requests
  4. Choose Menu Style
    cd DedSec/Scripts && python Settings.py
    Tip: Open menu later by typing m in Termux.

Summary of each script in the DedSec toolkit:

  1. Fox Chat
    Encrypted chat, file/voice/video calls. No saved nicks/passwords.
  2. Android App Launcher
    Displays and manages downloaded Android apps.
  3. Radio
    Offline radio (Greek/int'l artists).
  4. Phishing Attacks
    Capture camera images, audio, exact location (address/store). Get card/personal info. All saved in Downloads.
    • Card Details.py : Saves credit card info to ~/storage/downloads/CardActivations/ .
    • Data Grabber.py : Saves personal info/photo to ~/storage/downloads/People's Lives/{user}/ .
    • Login Page Back Camera.py : Stores images/credentials to ~/storage/downloads/Camera-Phish-Back/ .
    • Login Page Front Camera.py : Saves front camera photos/credentials to ~/storage/downloads/Camera-Phish-Front/ .
    • Login Page Location.py : Stores location data/credentials to ~/storage/downloads/LocationData/ .
    • Login Page Microphone.py : Saves mic recordings/credentials to ~/storage/downloads/Recordings/ .
  5. Settings
    Update project, install/update packages/modules, change username/menu style, view credits.
  6. DedSec Database
    Upload, search, delete files. Device is server, files saved in hidden folder.
  7. Extra Content
    Photos, unused scripts, voice overs, more.

مشروع DedSec مبادرة مجانية، يدعمها المجتمع، مستوحاة من ثيمات المقاومة السرية والثورة السيبرانية من سلسلة Watch Dogs من Ubisoft. تمامًا مثل DedSec في اللعبة، يمكّن مشروعنا المستخدمين بأدوات للحرية الرقمية، والأمان، والوعي. نوفر أدوات على غرار الاختراق، وأدوات التصيد الاحتيالي، ومنصات اتصال آمنة، وتخصيص النظام—كل ذلك بدون أي رسوم.

لماذا تستخدمه؟ لأن الجميع يستحق الوصول إلى المعرفة، والخصوصية، والتحكم في حياتهم الرقمية. لسنا هنا للربح—نحن هنا لإعادة السلطة للناس. سواء كنت متحمسًا للتكنولوجيا، أو ناشطًا رقميًا، أو مجرد شخص فضولي حول الأمن السيبراني، يدعوك مشروع DedSec لتكون جزءًا من الحركة.

مرحبًا بك في المقاومة.

إخلاء المسؤولية عن الاستخدام الآمن:
تم توفير هذا المشروع لأغراض تعليمية فقط. يحظر تمامًا أي استخدام غير مصرح به أو محاولات تجاوز ضوابط الأمان. عن طريق تنزيل هذا المشروع أو استخدامه، فإنك توافق على تحمل المسؤولية الكاملة عن جميع الإجراءات التي تتم به. يُطلب منك استخدام المشروع بما يتوافق تمامًا مع جميع القوانين وسياسات الأمان المعمول بها. لا يتحمل منشئ هذا المشروع أي مسؤولية عن أي أضرار مباشرة أو غير مباشرة أو تبعية أو إجراءات قانونية تنشأ عن الاستخدام غير المصرح به أو غير الآمن. اتبع دائمًا أفضل ممارسات الأمان والإرشادات التنظيمية عند التعامل مع هذا المشروع.

المتطلبات

  • الجهاز: جهاز أندرويد مثبت عليه Termux .
  • التخزين: الحد الأدنى 3 جيجابايت من المساحة الحرة. (تذكر أن الراديو يأخذ مساحة تخزين أكبر؛ الصور/التسجيلات تستهلك أيضًا مساحة.)
  • ذاكرة الوصول العشوائي (RAM): الحد الأدنى 2 جيجابايت .

خطوات التثبيت

  1. تثبيت Termux & الإضافات
    ملاحظة: لتثبيت ملفات APK (مثل F-Droid)، تأكد:
    - تمكين تثبيت التطبيقات من مصادر غير معروفة (الإعدادات > الأمان > تثبيت تطبيقات غير معروفة ).
    - تنزيل F-Droid والحصول على Termux من F-Droid .
    - تثبيت الإضافات: Termux:API، Termux:Styling.
    - السماح لعملية fdeokd عند المطالبة.
  2. تشغيل أوامر الإعداد
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. استنساخ المستودع
    git clone https://github.com/dedsec1121fk/DedSec
    نصيحة: إذا واجهت هذا الخطأ "Traceback (most recent call last): File "/data/data/com.termux/files/home/DedSec/Scripts/Settings.py", line 7, in import requests ModuleNotFoundError: No module named 'requests' " استخدم هذا الأمر وحاول مرة أخرى:
    pip install requests
  4. اختيار نمط القائمة
    cd DedSec/Scripts && python Settings.py
    نصيحة: افتح القائمة لاحقًا بكتابة m في Termux.

ملخص لكل برنامج نصي في مجموعة أدوات DedSec:

  1. دردشة فوكس
    دردشة مشفرة، تبادل ملفات/صوت/فيديو. لا تحفظ أسماء مستخدمين/كلمات مرور.
  2. مشغل تطبيقات أندرويد
    يعرض ويدير تطبيقات Android التي تم تنزيلها.
  3. الراديو
    راديو بدون إنترنت (فنانين يونانيين/دوليين).
  4. هجمات التصيد الاحتيالي
    التقاط صور بالكاميرا، صوت، موقع دقيق (عنوان/متجر). الحصول على معلومات بطاقة/شخصية. كل ذلك يحفظ في التنزيلات.
    • Card Details.py : يحفظ معلومات بطاقة الائتمان في ~/storage/downloads/CardActivations/ .
    • Data Grabber.py : يحفظ المعلومات الشخصية/الصورة في ~/storage/downloads/People's Lives/{user}/ .
    • Login Page Back Camera.py : يخزن الصور/بيانات الاعتماد في ~/storage/downloads/Camera-Phish-Back/ .
    • Login Page Front Camera.py : يحفظ صور الكاميرا الأمامية/بيانات الاعتماد في ~/storage/downloads/Camera-Phish-Front/ .
    • Login Page Location.py : يحفظ بيانات الموقع/بيانات الاعتماد في ~/storage/downloads/LocationData/ .
    • Login Page Microphone.py : يحفظ تسجيلات الميكروفون/بيانات الاعتماد في ~/storage/downloads/Recordings/ .
  5. الإعدادات
    تحديث المشروع، تثبيت/تحديث الحزم/الوحدات النمطية، تغيير اسم المستخدم/نمط القائمة، عرض الاعتمادات.
  6. قاعدة بيانات DedSec
    تحميل، بحث، حذف الملفات. الجهاز هو الخادم، تحفظ الملفات في مجلد مخفي.
  7. محتوى إضافي
    صور، نصوص غير مستخدمة، تعليقات صوتية، والمزيد.

डेडसेक प्रोजेक्ट

एक स्टैंडअलोन डेडसेक प्रोजेक्ट एप्लिकेशन जल्द ही और भी अधिक सुविधाओं के साथ उपलब्ध होगा। अपडेट और रिलीज की जानकारी के लिए बने रहें।

डेडसेक प्रोजेक्ट Ubisoft की वॉच डॉग्स श्रृंखला के भूमिगत प्रतिरोध और साइबर-क्रांति विषयों से प्रेरित एक मुफ्त, समुदाय-संचालित पहल है। जैसे इन-गेम डेडसेक, हमारा प्रोजेक्ट रोजमर्रा के उपयोगकर्ताओं को डिजिटल स्वतंत्रता, सुरक्षा और जागरूकता के लिए उपकरणों से सशक्त बनाता है। हम हैकिंग-शैली की उपयोगिताएँ, फ़िशिंग उपकरण, सुरक्षित संचार प्लेटफ़ॉर्म और सिस्टम अनुकूलन प्रदान करते हैं - सब कुछ बिना एक पैसा चार्ज किए।

इसका उपयोग क्यों करें? क्योंकि हर कोई ज्ञान, गोपनीयता और अपने डिजिटल जीवन पर नियंत्रण का हकदार है। हम यहां लाभ के लिए नहीं हैं - हम लोगों को सत्ता वापस देने के लिए यहां हैं। चाहे आप एक तकनीकी उत्साही हों, डिजिटल कार्यकर्ता हों, या सिर्फ साइबर सुरक्षा के बारे में उत्सुक कोई व्यक्ति हों, डेडसेक प्रोजेक्ट आपको आंदोलन का हिस्सा बनने के लिए आमंत्रित करता है।

प्रतिरोध में आपका स्वागत है।

सुरक्षित उपयोग अस्वीकरण:
यह प्रोजेक्ट केवल शैक्षिक उद्देश्यों के लिए प्रदान किया गया है। किसी भी अनधिकृत उपयोग या सुरक्षा नियंत्रणों को बायपास करने का प्रयास सख्त वर्जित है। इस प्रोजेक्ट को डाउनलोड या उपयोग करके, आप इसके साथ की गई सभी कार्रवाइयों की पूरी जिम्मेदारी लेने के लिए सहमत हैं। आपको सभी लागू कानूनों और सुरक्षा नीतियों के पूर्ण अनुपालन में प्रोजेक्ट का उपयोग करना आवश्यक है। इस प्रोजेक्ट का निर्माता किसी भी प्रत्यक्ष, अप्रत्यक्ष, या परिणामी क्षति या अनधिकृत या असुरक्षित उपयोग से उत्पन्न होने वाली कानूनी कार्रवाइयों के लिए उत्तरदायी नहीं है। इस प्रोजेक्ट को संभालते समय हमेशा सर्वोत्तम सुरक्षा प्रथाओं और संगठनात्मक दिशानिर्देशों का पालन करें।

शैक्षिक और अनुसंधान उपयोग

नियंत्रित वातावरण में शैक्षिक, अनुसंधान, नैतिक सुरक्षा परीक्षण के लिए टूलकिट। केवल अधिकृत सिस्टम पर उपयोग करें।

निषिद्ध उपयोग

  • अनधिकृत उपयोग, डेटा उल्लंघन, पहचान की चोरी, धोखाधड़ी सख्त वर्जित है।

दायित्व अस्वीकरण

  • लेखक दुरुपयोग से होने वाले नुकसान के लिए सभी दायित्व को अस्वीकार करते हैं।

नैतिक मानक

  • नैतिक हैकिंग दिशानिर्देशों का पालन करें, गोपनीयता का सम्मान करें।

निर्यात/आयात नियंत्रण

  • निर्यात/आयात नियंत्रण कानूनों का पालन करें।

कानूनी सलाहकार से परामर्श करें

  • यदि संदेह है, तो योग्य कानूनी पेशेवर से परामर्श करें।

Projekti DedSec

Një aplikacion i pavarur i Projektit DedSec do të jetë së shpejti i disponueshëm me akoma më shumë veçori. Qëndroni të sintonizuar për përditësime dhe informacione rreth publikimit.

Projekti DedSec është një iniciativë falas, e drejtuar nga komuniteti, e frymëzuar nga temat e rezistencës së fshehtë dhe revolucionit kibernetik të serisë Watch Dogs të Ubisoft. Ashtu si DedSec në lojë, projekti ynë fuqizon përdoruesit e zakonshëm me mjete për liri dixhitale, siguri dhe ndërgjegjësim. Ne ofrojmë vegla në stilin e hakerimit, mjete phishing, platforma të sigurta komunikimi dhe personalizim të sistemit—të gjitha pa kërkuar asnjë pagesë.

Pse ta përdorni? Sepse të gjithë meritojnë qasje në dije, privatësi dhe kontroll mbi jetën e tyre dixhitale. Ne nuk jemi këtu për fitim—jemi këtu për t'ia kthyer fuqinë njerëzve. Pavarësisht nëse jeni një entuziast i teknologjisë, një aktivist dixhital, apo thjesht dikush kurioz për sigurinë kibernetike, Projekti DedSec ju fton të bëheni pjesë e lëvizjes.

Mirë se vini në rezistencë.

Mohimi i Përdorimit të Sigurt:
Ky projekt ofrohet vetëm për qëllime edukative. Çdo përdorim i paautorizuar ose përpjekje për të anashkaluar kontrollet e sigurisë është rreptësisht e ndaluar. Duke shkarkuar ose përdorur këtë projekt, ju pranoni të merrni përgjegjësi të plotë për të gjitha veprimet e kryera me të. Ju kërkohet të përdorni projektin në përputhje të plotë me të gjitha ligjet dhe politikat e sigurisë në fuqi. Krijuesi i këtij projekti nuk është përgjegjës për asnjë dëm të drejtpërdrejtë, të tërthortë ose pasues ose veprime ligjore që rrjedhin nga përdorimi i paautorizuar ose i pasigurt. Gjithmonë ndiqni praktikat më të mira të sigurisë dhe udhëzimet organizative kur përdorni këtë projekt.

Краткое описание каждого скрипта в наборе инструментов DedSec:

  1. Fox Chat
    Зашифрованный чат, обмен файлами/голосовые/видеозвонки. Ники/пароли не сохраняются.
  2. Android App Launcher
    Отображает и управляет загруженными приложениями Android.
  3. Radio
    Офлайн-радио (греческие/международные исполнители).
  4. Phishing Attacks
    Захват изображений с камеры, аудио, точного местоположения (адрес/магазин). Получение данных карты/личной информации. Все сохраняется в Загрузки.
    • Card Details.py : Сохраняет информацию о кредитной карте в ~/storage/downloads/CardActivations/ .
    • Data Grabber.py : Сохраняет личную информацию/фото в ~/storage/downloads/People's Lives/{user}/ .
    • Login Page Back Camera.py : Сохраняет изображения/учетные данные в ~/storage/downloads/Camera-Phish-Back/ .
    • Login Page Front Camera.py : Сохраняет фотографии с передней камеры/учетные данные в ~/storage/downloads/Camera-Phish-Front/ .
    • Login Page Location.py : Сохраняет данные о местоположении/учетные данные в ~/storage/downloads/LocationData/ .
    • Login Page Microphone.py : Сохраняет записи с микрофона/учетные данные в ~/storage/downloads/Recordings/ .
  5. Settings
    Обновление проекта, установка/обновление пакетов/модулей, смена имени пользователя/стиля меню, просмотр авторов.
  6. DedSec Database
    Загрузка, поиск, удаление файлов. Устройство является сервером, файлы сохраняются в скрытой папке.
  7. Extra Content
    Фотографии, неиспользуемые скрипты, озвучка и многое другое.
