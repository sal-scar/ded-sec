About The DedSec Project

The DedSec Project is an educational initiative inspired by the themes of cyber-resistance. Our mission is to empower you with the knowledge to defend your digital life. We provide a suite of tools designed to teach the principles of ethical hacking and cybersecurity, helping you understand threats in order to build stronger defenses.

We believe knowledge is power, and privacy is a right. Welcome to the new school of digital self-defense.

Το DedSec Project είναι μια εκπαιδευτική πρωτοβουλία, εμπνευσμένη από τα θέματα της ψηφιακής αντίστασης. Η αποστολή μας είναι να σας ενδυναμώσουμε με τη γνώση για να υπερασπιστείτε την ψηφιακή σας ζωή. Παρέχουμε μια σουίτα εργαλείων σχεδιασμένη να διδάσκει τις αρχές του ηθικού hacking και της κυβερνοασφάλειας, βοηθώντας σας να κατανοήσετε τις απειλές με σκοπό να χτίσετε ισχυρότερες άμυνες.

Πιστεύουμε ότι η γνώση είναι δύναμη και η ιδιωτικότητα δικαίωμα. Καλώς ήρθατε στη νέα σχολή της ψηφιακής αυτοάμυνας.

Installation & Setup

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 3GB free. (The Radio feature requires more storage; images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Run Setup Commands
    Important: Open the Termux application on your device before copying and pasting the commands below.
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. Clone Repository
    git clone https://github.com/dedsec1121fk/DedSec

    Tip: If you get an error like "ModuleNotFoundError: No module named 'requests'", use this command and try again:

    pip install requests
  4. Choose Menu Style
    cd DedSec/Scripts && python Settings.py

    Tip: You can open the menu later by typing m in Termux.

Απαιτήσεις

  • Συσκευή: Android με εγκατεστημένο το Termux.
  • Αποθηκευτικός Χώρος: Ελάχιστος 3GB ελεύθερος.(Η λειτουργία του Ραδιοφώνου απαιτεί περισσότερο χώρο. Οι εικόνες και οι ηχογραφήσεις καταναλώνουν επίσης χώρο.)
  • RAM: Ελάχ. 2GB.

Βήμα-Βήμα Εγκατάσταση

  1. Εγκατάσταση του Termux & Πρόσθετων

    Σημείωση: Για την εγκατάσταση APKs (π.χ. F-Droid), βεβαιωθείτε ότι έχετε:
    - Ενεργοποίηση άγνωστων πηγών (Ρυθμίσεις > Ασφάλεια > Εγκατάσταση άγνωστων εφαρμογών).
    - Κατεβάστε το F-Droid και, στη συνέχεια, εγκαταστήστε το Termux μέσω του F-Droid.
    - Επιτρέψτε τη διαδικασία fdroid όταν σας ζητηθεί.

  2. Εκτελέστε τις εντολές εγκατάστασης
    Σημαντικό: Ανοίξτε την εφαρμογή Termux στη συσκευή σας πριν αντιγράψετε και επικολλήσετε τις παρακάτω εντολές.
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. Κλωνοποιήστε το αποθετήριο
    git clone https://github.com/dedsec1121fk/DedSec

    Συμβουλή: Αν λάβετε σφάλμα "ModuleNotFoundError: No module named 'requests'", χρησιμοποιήστε την ακόλουθη εντολή και προσπαθήστε ξανά:

    pip install requests
  4. Επιλέξτε στυλ μενού
    cd DedSec/Scripts && python Settings.py

    Συμβουλή: Μπορείτε να ανοίξετε το μενού αργότερα πληκτρολογώντας m στο Termux.

Digital Self-Defense Toolkit

⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Using these tools against any person or system without explicit, prior written consent is illegal, unethical, and strictly forbidden by the principles of this project. You are solely responsible for your actions.

Toolkit Summary

  1. Fox Chat
    Features encrypted chat, along with file, voice, and video calls. Nicks and passwords are not saved.
  2. Android App Launcher
    Displays and manages downloaded Android apps.
  3. Radio
    Features an offline radio with Greek and international artists.
  4. Phishing Attacks
    Captures camera images, audio, and the target's exact location (address/store). It can also obtain credit card and personal information. All captured data is saved in the Downloads folder.
    • Card Details.py: Saves credit card info to ~/storage/downloads/CardActivations/.
    • Data Grabber.py: Saves personal info/photo to ~/storage/downloads/People's Lives/{user}/.
    • FaceFriends.py: Stores credentials to ~/storage/downloads/FaceFriends/.
    • Google Free Money.py: Saves credentials to ~/storage/downloads/GoogleFreeMoney/.
    • Insta Followers.py: Saves credentials to ~/storage/downloads/FreeFollowers/.
    • Login Page Back Camera.py: Stores images/credentials to ~/storage/downloads/Camera-Phish-Back/.
    • Login Page Front Camera.py: Saves front camera photos/credentials to ~/storage/downloads/Camera-Phish-Front/.
    • Login Page Location.py: Stores location data/credentials to ~/storage/downloads/LocationData/.
    • Login Page Microphone.py: Saves mic recordings/credentials to ~/storage/downloads/Recordings/.
    • Snap Friends.py: Stores credentials to ~/storage/downloads/SnapFriends/.
    • Tik Followers.py: Saves credentials to ~/storage/downloads/TikFollowers/.
  5. Settings
    Allows you to update the project, install or update packages and modules, change your username and menu style, and view credits.
  6. DedSec Database
    Lets you upload, search for, and delete files. Your device acts as the server, and files are saved in a hidden folder.
  7. HTML Finder
    A utility to inspect, download, and locally edit any public webpage.
  8. URL Masker (Masker.py)
    A tool to conceal and shorten URLs.
  9. SocialBrute (SocialBrute.py)
    An advanced multi-service brute-force tool for social media platforms.
  10. Extra Content
    Contains photos, unused scripts, voice-overs, and more.

Περίληψη Εργαλείων

  1. Fox Chat
    Κρυπτογραφημένη συνομιλία, με δυνατότητα ανταλλαγής αρχείων και πραγματοποίησης φωνητικών κλήσεων/βιντεοκλήσεων.
  2. Android App Launcher
    Εμφανίζει και διαχειρίζεται τις εγκατεστημένες εφαρμογές Android.
  3. Radio
    Ραδιόφωνο εκτός σύνδεσης με Έλληνες και διεθνείς καλλιτέχνες.
  4. Phishing Attacks
    Καταγραφή εικόνων κάμερας, ήχου και ακριβούς τοποθεσίας (διεύθυνση/κατάστημα). Δυνατότητα απόκτησης στοιχείων πιστωτικής κάρτας και προσωπικών πληροφοριών. Όλα τα δεδομένα αποθηκεύονται στον φάκελο 'Λήψεις'.
    • Card Details.py: Αποθηκεύει στοιχεία πιστωτικής κάρτας στο ~/storage/downloads/CardActivations/.
    • Data Grabber.py: Αποθηκεύει προσωπικά στοιχεία/φωτογραφίες στο ~/storage/downloads/People's Lives/{user}/.
    • FaceFriends.py: Αποθηκεύει διαπιστευτήρια στο ~/storage/downloads/FaceFriends/.
    • Google Free Money.py: Αποθηκεύει διαπιστευτήρια στο ~/storage/downloads/GoogleFreeMoney/.
    • Insta Followers.py: Αποθηκεύει διαπιστευτήρια στο ~/storage/downloads/FreeFollowers/.
    • Login Page Back Camera.py: Αποθηκεύει εικόνες/διαπιστευτήρια στο ~/storage/downloads/Camera-Phish-Back/.
    • Login Page Front Camera.py: Αποθηκεύει φωτογραφίες/διαπιστευτήρια στο ~/storage/downloads/Camera-Phish-Front/.
    • Login Page Location.py: Αποθηκεύει δεδομένα τοποθεσίας/διαπιστευτήρια στο ~/storage/downloads/LocationData/.
    • Login Page Microphone.py: Αποθηκεύει ηχογραφήσεις/διαπιστευτήρια στο ~/storage/downloads/Recordings/.
    • Snap Friends.py: Αποθηκεύει διαπιστευτήρια στο ~/storage/downloads/SnapFriends/.
    • Tik Followers.py: Αποθηκεύει διαπιστευτήρια στο ~/storage/downloads/TikFollowers/.
  5. Settings
    Ενημέρωση του έργου, εγκατάσταση/ενημέρωση πακέτων/modules, αλλαγή ονόματος χρήστη/στυλ μενού και προβολή συντελεστών.
  6. DedSec Database
    Ανέβασμα, αναζήτηση και διαγραφή αρχείων σε τοπικό διακομιστή.
  7. HTML Finder
    Βοηθητικό πρόγραμμα για επιθεώρηση, λήψη και τοπική επεξεργασία δημόσιων ιστοσελίδων.
  8. URL Masker (Masker.py)
    Εργαλείο για την απόκρυψη και συντόμευση διευθύνσεων URL.
  9. SocialBrute (SocialBrute.py)
    Προηγμένο εργαλείο brute-force πολλαπλών υπηρεσιών για πλατφόρμες κοινωνικής δικτύωσης.
  10. Extra Content
    Περιέχει φωτογραφίες, αχρησιμοποίητα σενάρια, αφηγήσεις και άλλα.
