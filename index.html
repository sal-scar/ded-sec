DedSec Project

A standalone DedSec Project application will be soon available with even more features. Stay tuned for updates and release info.

The DedSec Project is a free, community-driven initiative inspired by the underground resistance and cyber-revolution themes of Ubisoft’s Watch Dogs series. Just like the in-game DedSec, our project empowers everyday users with tools for digital freedom, security, and awareness. We provide hacking-style utilities, phishing tools, secure communication platforms, and system customization—all without charging a cent.

Why use it? Because everyone deserves access to knowledge, privacy, and control over their digital life. We're not here for profit—we're here to give power back to the people. Whether you're a tech enthusiast, digital activist, or just someone curious about cybersecurity, the DedSec Project invites you to be part of the movement.

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 3GB free. (Radio takes more storage; images/recordings also consume space.)
  • RAM: Min 2GB .

Step-by-Step Installation

  1. Install Termux & Add-Ons
    Note: To install APKs (e.g., F-Droid), ensure:
    - Enabled unknown sources (Settings > Security > Install Unknown Apps ).
    - Download F-Droid, get Termux from F-Droid .
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow fdroid process when prompted.
  2. Run Setup Commands
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. Clone Repository
    git clone https://github.com/dedsec1121fk/DedSec
    Tip: If you get his error "Traceback (most recent call last): File "/data/data/com.termux/files/home/DedSec/Scripts/Settings.py", line 7, in import requests ModuleNotFoundError: No module named 'requests' " use this command and try again:
    pip install requests
  4. Choose Menu Style
    cd DedSec/Scripts && python Settings.py
    Tip: Open menu later by typing m in Termux.

Summary of each script in the DedSec toolkit:

  1. Fox Chat
    Encrypted chat, file/voice/video calls. No saved nicks/passwords.
  2. Android App Launcher
    Displays and manages downloaded Android apps.
  3. Radio
    Offline radio (Greek/int'l artists).
  4. Phishing Attacks
    Capture camera images, audio, exact location (address/store). Get card/personal info. All saved in Downloads.
    • Card Details.py : Saves credit card info to ~/storage/downloads/CardActivations/ .
    • Data Grabber.py : Saves personal info/photo to ~/storage/downloads/People's Lives/{user}/ .
    • Login Page Back Camera.py : Stores images/credentials to ~/storage/downloads/Camera-Phish-Back/ .
    • Login Page Front Camera.py : Saves front camera photos/credentials to ~/storage/downloads/Camera-Phish-Front/ .
    • Login Page Location.py : Stores location data/credentials to ~/storage/downloads/LocationData/ .
    • Login Page Microphone.py : Saves mic recordings/credentials to ~/storage/downloads/Recordings/ .
  5. Settings
    Update project, install/update packages/modules, change username/menu style, view credits.
  6. DedSec Database
    Upload, search, delete files. Device is server, files saved in hidden folder.
  7. HTML Finder
    A utility to inspect, download, and locally edit any public webpage. It automatically downloads the main HTML file and its assets (CSS, JS, images, icons, media), saves them in a local folder, and opens the page in a terminal editor. Files are stored under:
    • ~/storage/downloads/<website_name>/index.html
    • Assets are organized into subfolders like css/ , js/ , images/ , etc.
    • Supports editing via nano , vim , or any CLI editor.
    • Optional browser preview with termux-open-url .
  8. Extra Content
    Photos, unused scripts, voice overs, more.

    - Κατεβάστε το F-Droid, εγκαταστήστε το Termux μέσω F-Droid .
    - Εγκαταστήστε πρόσθετα: Termux:API, Termux:Styling.
    - Επιτρέψτε τη διαδικασία fdroid όταν σας ζητηθεί.
  2. Εκτελέστε τις εντολές εγκατάστασης
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install python git fzf nodejs openssh nano jq wget unzip curl proot openssl aapt rust cloudflared
    pip install flask blessed flask-socketio werkzeug requests datetime geopy pydub pycryptodome mutagen rust cryptography phonenumbers pycountry
    pkg install termux-api
  3. Κλωνοποιήστε το αποθετήριο
    git clone https://github.com/dedsec1121fk/DedSec
    Συμβουλή: Αν λάβετε το σφάλμα "Traceback (most recent call last): File "/data/data/com.termux/files/home/DedSec/Scripts/Settings.py", line 7, in import requests ModuleNotFoundError: No module named 'requests' " χρησιμοποιήστε την ακόλουθη εντολή και προσπαθήστε ξανά:
    pip install requests
  4. Επιλέξτε στυλ μενού
    cd DedSec/Scripts && python Settings.py
    Συμβουλή: Ανοίξτε το μενού αργότερα πληκτρολογώντας m στο Termux.

Περίληψη κάθε σεναρίου στο DedSec toolkit:

  1. Fox Chat
    Κρυπτογραφημένη συνομιλία, ανταλλαγή αρχείων, φωνητικών κλήσεων/βιντεοκλήσεων. Δεν αποθηκεύονται ψευδώνυμα ή κωδικοί πρόσβασης.
  2. Android App Launcher
    Εμφανίζει και διαχειρίζεται εγκατεστημένες εφαρμογές Android.
  3. Radio
    Offline ραδιόφωνο (Έλληνες/διεθνείς καλλιτέχνες).
  4. Phishing Attacks
    Λήψη εικόνων κάμερας, ήχου, ακριβούς τοποθεσίας (διεύθυνση/κατάστημα). Απόκτηση στοιχείων κάρτας και προσωπικών πληροφοριών. Όλα αποθηκεύονται στον φάκελο Downloads.
    • Card Details.py : Αποθηκεύει στοιχεία πιστωτικής κάρτας στο ~/storage/downloads/CardActivations/ .
    • Data Grabber.py : Αποθηκεύει προσωπικά στοιχεία/φωτογραφίες στο ~/storage/downloads/People's Lives/{user}/ .
    • Login Page Back Camera.py : Αποθηκεύει εικόνες/διαπιστευτήρια στο ~/storage/downloads/Camera-Phish-Back/ .
    • Login Page Front Camera.py : Αποθηκεύει φωτογραφίες μπροστινής κάμερας/διαπιστευτήρια στο ~/storage/downloads/Camera-Phish-Front/ .
    • Login Page Location.py : Αποθηκεύει δεδομένα τοποθεσίας/διαπιστευτήρια στο ~/storage/downloads/LocationData/ .
    • Login Page Microphone.py : Αποθηκεύει ηχογραφήσεις μικροφώνου/διαπιστευτήρια στο ~/storage/downloads/Recordings/ .
  5. Settings
    Ενημέρωση έργου, εγκατάσταση/ενημέρωση πακέτων/modules, αλλαγή ονόματος χρήστη/στυλ μενού, προβολή συντελεστών.
  6. DedSec Database
    Ανέβασμα, αναζήτηση, διαγραφή αρχείων. Η συσκευή λειτουργεί ως διακομιστής, τα αρχεία αποθηκεύονται σε κρυφό φάκελο.
  7. HTML Finder
    Ένα βοηθητικό πρόγραμμα για την επιθεώρηση, λήψη και τοπική επεξεργασία οποιασδήποτε δημόσιας ιστοσελίδας. Αυτόματα κατεβάζει το κύριο αρχείο HTML και τα στοιχεία της (CSS, JS, εικόνες, εικονίδια, και μέσα), τα αποθηκεύει σε τοπικό φάκελο και ανοίγει τη σελίδα σε έναν επεξεργαστή τερματικού. Τα αρχεία αποθηκεύονται στις ακόλουθες διαδρομές:
    • ~/storage/downloads/<website_name>/index.html
    • Τα στοιχεία οργανώνονται σε υποφακέλους όπως css/ , js/ , images/ , κ.λπ.
    • Υποστηρίζει επεξεργασία μέσω nano , vim , ή οποιουδήποτε επεξεργαστή CLI.
    • Προαιρετική προεπισκόπηση προγράμματος περιήγησης με termux-open-url .
  8. Extra Content
    Φωτογραφίες, αχρησιμοποίητα scripts, αφηγήσεις, κ.α.
