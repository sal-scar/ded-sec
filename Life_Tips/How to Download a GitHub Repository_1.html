Πώς να Κατεβάσετε ένα Αποθετήριο GitHub

Ένας αναλυτικός οδηγός για αρχάριους για το πώς να κατεβάσετε όλα τα αρχεία ενός project από το GitHub στον υπολογιστή ή τη συσκευή σας Android.

Σε Υπολογιστή (Windows, Mac, Linux)

  1. Πηγαίνετε στη σελίδα του project στο GitHub. Πάνω από τη λίστα αρχείων, κάντε κλικ στο πράσινο κουμπί <> Code.
  2. Από το αναπτυσσόμενο μενού, κάντε κλικ στο Download ZIP. Η λήψη θα ξεκινήσει.
  3. Βρείτε το αρχείο που κατεβάσατε. Είναι σχεδόν πάντα στον φάκελό σας 'Λήψεις'.

    Σε Windows: Ανοίξτε την 'Εξερεύνηση αρχείων' (το κίτρινο εικονίδιο φακέλου στη γραμμή εργασιών) και κάντε κλικ στις 'Λήψεις' στην αριστερή πλευρά.

    Σε Mac: Ανοίξτε το 'Finder' (το εικονίδιο με το χαμογελαστό πρόσωπο στο Dock) και κάντε κλικ στις 'Λήψεις' στην πλαϊνή μπάρα.

  4. Το αρχείο θα είναι ένα αρχείο .zip. Κάντε δεξί κλικ πάνω του και επιλέξτε 'Εξαγωγή όλων...' ή 'Αποσυμπίεση' για να δημιουργήσετε έναν κανονικό φάκελο με όλα τα αρχεία του project.

Pro Tip: Πριν εκτελέσετε οποιοδήποτε script από ένα αποθετήριο που κατεβάσατε, είναι καλή πρακτική ασφαλείας να επιθεωρείτε τον κώδικα για οτιδήποτε ύποπτο, ειδικά αν δεν εμπιστεύστε πλήρως την πηγή.

Σε Συσκευή Android (Ο Εύκολος Τρόπος)

  1. Στο πρόγραμμα περιήγησής σας, πηγαίνετε στη σελίδα του project στο GitHub. Δεν χρειάζονται ειδικά κόλπα!
  2. Ακριβώς όπως και στον υπολογιστή, κοιτάξτε πάνω από τη λίστα αρχείων και πατήστε το πράσινο κουμπί <> Code.
  3. Θα εμφανιστεί ένα μενού. Στο κάτω μέρος αυτού του μενού, πατήστε τον σύνδεσμο 'Download ZIP'. Η λήψη θα ξεκινήσει.
  4. Βρείτε το αρχείο που κατεβάσατε. Πού βρίσκεται;

    Ο ευκολότερος τρόπος: Τραβήξτε προς τα κάτω τη γραμμή ειδοποιήσεων από την κορυφή της οθόνης. Πατήστε στην ειδοποίηση 'Η λήψη ολοκληρώθηκε'.

    Ο σίγουρος τρόπος: Ανοίξτε την εφαρμογή 'Αρχεία' ή 'Τα αρχεία μου' του τηλεφώνου σας (ψάξτε για ένα εικονίδιο φακέλου). Μέσα, πηγαίνετε στον φάκελο 'Λήψεις'.

  5. Πατήστε στο αρχείο .zip. Το τηλεφώνό σας θα σας ζητήσει να το 'Εξάγετε'. Πατήστε 'Εξαγωγή' και θα δημιουργηθεί ένας νέος φάκελος με όλα τα αρχεία του project.

Pro Tip: Μετά την εξαγωγή, χρησιμοποιήστε έναν διαχειριστή αρχείων για να μετακινήσετε τον φάκελο του project σε μια πιο οργανωμένη τοποθεσία. Εξετάστε το ενδεχόμενο να χρησιμοποιήσετε μια εφαρμογή ασφαλείας για να σαρώσετε νέα αρχεία εάν δεν είστε σίγουροι για την προέλευσή τους.
