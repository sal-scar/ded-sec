How to Download a GitHub Repository

A detailed guide for beginners on how to download all the files from a project on GitHub to your computer or Android device.

On Desktop (Windows, Mac, Linux)

  1. Go to the GitHub project page. Above the file list, click the green <> Code button.
  2. From the dropdown menu, click on Download ZIP. The download will begin.
  3. Find the downloaded file. It's almost always in your 'Downloads' folder.

    On Windows: Open File Explorer (the yellow folder icon on your taskbar) and click 'Downloads' on the left side.

    On Mac: Open Finder (the smiling face icon in your Dock) and click 'Downloads' in the sidebar.

  4. The file will be a .zip file. Right-click it and choose 'Extract All...' or 'Unzip' to create a normal folder with all the project files.

On an Android Device (The Easy Way)

  1. In your browser, go to the GitHub project page. No special tricks needed!
  2. Just like on desktop, look above the file list and tap the green <> Code button.
  3. A menu will appear. At the bottom of this menu, tap the 'Download ZIP' link. The download will start.
  4. Find the downloaded file. Where is it?

    Easiest way: Pull down your notification bar from the top of the screen. Tap on the 'Download complete' notification.

    Sure way: Open your phone's 'Files' or 'My Files' app (look for a folder icon). Inside, go to the 'Downloads' folder.

  5. Tap on the .zip file. Your phone will ask you to 'Extract' it. Tap 'Extract', and it will create a new folder with all the project files.
