How to Download a GitHub Repository

Here are the two primary methods for downloading a project from GitHub onto your computer or Android device via Termux.

On PC / Desktop

  1. Using Git (Recommended): This is the best method if you plan to get updates. You need to have Git installed on your system.
    1. Go to the GitHub repository page.
    2. Click the green 'Code' button.
    3. Copy the HTTPS URL provided.
    4. Open a terminal or command prompt and run the following command:
    git clone COPIED_URL_HERE
  2. Downloading the ZIP file: This method is simpler but won't allow you to easily update the project later.
    1. Go to the GitHub repository page.
    2. Click the green 'Code' button.
    3. Click 'Download ZIP'.
    4. Save the file and then extract it to your desired location.

On Android (Using Termux)

  1. Install Git: If you haven't already, open Termux and install Git.
    pkg install git
  2. Clone the Repository: Copy the HTTPS URL from the GitHub page (just like in the PC method) and run the `git clone` command in Termux.
    git clone COPIED_URL_HERE

    The repository files will be downloaded into a new folder in your Termux home directory. You can list the files with `ls` to see it.

Πώς να Κατεβάσετε ένα Αποθετήριο GitHub

Παρακάτω είναι οι δύο κύριες μέθοδοι για να κατεβάσετε ένα έργο από το GitHub στον υπολογιστή σας ή στη συσκευή σας Android μέσω του Termux.

Σε PC / Desktop

  1. Με χρήση Git (Προτείνεται): Αυτή είναι η καλύτερη μέθοδος αν σκοπεύετε να λαμβάνετε ενημερώσεις. Πρέπει να έχετε εγκατεστημένο το Git στο σύστημά σας.
    1. Πηγαίνετε στη σελίδα του αποθετηρίου στο GitHub.
    2. Κάντε κλικ στο πράσινο κουμπί 'Code'.
    3. Αντιγράψτε το HTTPS URL που παρέχεται.
    4. Ανοίξτε ένα τερματικό ή γραμμή εντολών και εκτελέστε την παρακάτω εντολή:
    git clone ΑΝΤΙΓΡΑΜΜΕΝΟ_URL_ΕΔΩ
  2. Λήψη αρχείου ZIP: Αυτή η μέθοδος είναι απλούστερη αλλά δεν σας επιτρέπει να ενημερώσετε εύκολα το έργο αργότερα.
    1. Πηγαίνετε στη σελίδα του αποθετηρίου στο GitHub.
    2. Κάντε κλικ στο πράσινο κουμπί 'Code'.
    3. Κάντε κλικ στο 'Download ZIP'.
    4. Αποθηκεύστε το αρχείο και στη συνέχεια αποσυμπιέστε το στην τοποθεσία που επιθυμείτε.

Σε Android (Με χρήση Termux)

  1. Εγκατάσταση Git: Αν δεν το έχετε κάνει ήδη, ανοίξτε το Termux και εγκαταστήστε το Git.
    pkg install git
  2. Κλωνοποίηση του Αποθετηρίου: Αντιγράψτε το HTTPS URL από τη σελίδα του GitHub (όπως και στη μέθοδο για PC) και εκτελέστε την εντολή `git clone` στο Termux.
    git clone ΑΝΤΙΓΡΑΜΜΕΝΟ_URL_ΕΔΩ

    Τα αρχεία του αποθετηρίου θα ληφθούν σε έναν νέο φάκελο στον αρχικό κατάλογο του Termux. Μπορείτε να δείτε τη λίστα των αρχείων με την εντολή `ls` για να τον βρείτε.
