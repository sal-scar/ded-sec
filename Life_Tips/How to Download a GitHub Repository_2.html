How to download a GitHub repository — Android & PC

Step-by-step, simple instructions to get a copy of any GitHub repo on your Android device or on a computer.

On Android (English)

  1. Using a web browser (Download ZIP):
    1. Open the repository page on github.com in your browser (Chrome, Firefox, etc.).
    2. Tap the green Code button near the top-right of the file list.
    3. Choose Download ZIP. The repository will download as a ZIP file to your device's Downloads folder.
    4. Open your Files app to extract the ZIP.
  2. Using the official GitHub app:
    1. Install the GitHub app from the Play Store.
    2. Open the repo inside the app. Use "Open in browser" and follow the browser steps above.
  3. Using a Git client on Android:
    1. Install Termux or MGit.
    2. With Termux: run pkg update && pkg install git, then git clone https://github.com/USERNAME/REPOSITORY.git.
    3. With MGit: use the Clone option and paste the repo URL.

If a repo is private you'll need to sign in or set up SSH keys.

Σε Android (Ελληνικά)

  1. Μέσω προγράμματος περιήγησης (Λήψη ZIP):
    1. Άνοιξε τη σελίδα του αποθετηρίου στο github.com με τον browser σου.
    2. Πάτησε το πράσινο κουμπί Code πάνω δεξιά.
    3. Επίλεξε Download ZIP. Θα κατέβει στον φάκελο “Λήψεις”.
    4. Χρησιμοποίησε την εφαρμογή “Αρχεία” για να αποσυμπιέσεις το ZIP.
  2. Με την εφαρμογή GitHub:
    1. Κατέβασε το GitHub app από το Play Store.
    2. Άνοιξε το repo στην εφαρμογή. Χρησιμοποίησε το "Άνοιγμα σε browser" και ακολούθησε τα βήματα για browser.
  3. Με Git client σε Android:
    1. Εγκατάστησε το Termux ή το MGit.
    2. Με Termux: εκτέλεσε pkg update && pkg install git, μετά git clone https://github.com/USERNAME/REPOSITORY.git.
    3. Με MGit: επίλεξε Clone και επικόλλησε το URL του repo.

Για ιδιωτικά repos χρειάζεται σύνδεση ή SSH keys.

On PC (English)

  1. Download ZIP:
    1. Open the repository page on github.com.
    2. Click Code → Download ZIP.
    3. Extract the ZIP file on your computer.
  2. Using Git:
    1. Install Git on your system.
    2. Open terminal and run: 
      git clone https://github.com/USERNAME/REPOSITORY.git
    3. Use git pull to update later.

Σε Υπολογιστή (Ελληνικά)

  1. Λήψη ZIP:
    1. Άνοιξε το repo στο github.com.
    2. Κάνε κλικ στο Code → Download ZIP.
    3. Αποσυμπίεσε το ZIP στον υπολογιστή σου.
